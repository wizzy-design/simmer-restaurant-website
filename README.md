**Simm3r Restaurant & Café** is a premium, high-end dining sanctuary located in Jos, Plateau State. Built with cutting-edge web technologies, this platform offers a seamless user experience for exploring exquisite cuisines, making reservations, and discovering culinary excellence.

[**🌐 Live Demo**](https://simmer-restaurant.vercel.app) | [**📄 Documentation**](https://github.com/wizzy-design/simmer-restaurant-website#readme)

---

## ✨ Key Features

- **🎯 Sophisticated Reservation System**: Interactive sidebar and modal-based booking experience with real-time state management.
- **📜 Dynamic Food Menu**: A beautifully categorized menu featuring high-quality imagery and detailed descriptions.
- **⚡ Performance Optimized**: Sub-second load times using Next.js App Router, image optimization, and pre-connecting to heavy origins.
- **🔍 Advanced SEO**: Full Schema.org integration (Restaurant, Organization), dynamic metadata, and automated sitemaps/robots.ts.
- **🖼️ Media Excellence**: Leveraging Cloudinary for high-performance image and video delivery, including adaptive bitrate streaming.
- **📱 Fluid Responsiveness**: A mobile-first design philosophy using Tailwind CSS 4 for a perfect experience on all devices.
- **🎭 Engaging Animations**: Subtle micro-interactions and smooth transitions powered by Framer Motion.

---

## 🚀 Tech Stack

- **Core**: Next.js 15+ (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS 4 (Modern utility-first CSS)
- **Animation**: Motion (Framer Motion)
- **State Management**: React Context API
- **Images/Video**: Next-Cloudinary (Optimized media delivery)
- **Forms/Email**: Resend (Serverless email handling)
- **UI Components**: Lucide React, React Icons, React Fast Marquee
- **Analytics**: Vercel Analytics

## 🛠️ Installation & Setup

### Prerequisites
- Node.js 18+ 
- npm / yarn / pnpm

### Steps
1. **Clone the repository**
   ```bash
   git clone https://github.com/wizzy-design/simmer-restaurant-website.git
   cd simmer-restaurant-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Variables**
   Create a `.env.local` file in the root directory and add your credentials:
   ```env
   NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your_cloud_name
   CLOUDINARY_API_KEY=your_api_key
   CLOUDINARY_API_SECRET=your_api_secret
   RESEND_API_KEY=your_resend_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view the application.
